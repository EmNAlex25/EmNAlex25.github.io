This 3D printing project was a struggle for me, mostly attempting to create a model that would be able to be printed. I ended up going super simple, taking a model of the Sundial Bridge in Redding, CA where I grew up and printing it without many details. Unfortunately the supports were too tiny to print, so I had to improvise with string and glue. I detailed paint to make up for the simplified model.
